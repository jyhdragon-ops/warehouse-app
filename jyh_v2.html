📖 Google Sheets 설정 가이드

아래 순서대로 따라하면 5분 안에 연동됩니다.

1
Google Sheets를 새로 만들고, 첫 번째 시트 이름을 입출고대장으로 변경합니다.
2
1행에 헤더를 입력합니다:
ID 날짜 구분 품목 수량 단위 거래처 담당자 비고
3
상단 메뉴 확장 프로그램 → Apps Script를 클릭합니다.
4
아래 코드를 붙여넣고 저장(💾)합니다.
5
배포 → 새 배포 → 유형: 웹 앱 → 액세스: 모든 사용자 → 배포
6
생성된 웹 앱 URL을 복사해서 이 앱의 Sheets 연동 URL 칸에 붙여넣고 저장합니다.
7
연결 테스트 버튼으로 확인 후 완료!
